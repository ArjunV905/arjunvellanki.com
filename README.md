## Installation & Set Up

1. Install dependencies

   ```sh
   npm install
   ```

2. Start the development server (runs on [localhost:3000](http://localhost:3000))

   ```sh
   npm run dev
   ```

3. Generate a full static production build

   ```sh
   next build
   ```
